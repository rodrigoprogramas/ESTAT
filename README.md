# ISEP LEI Statistics (ESTAT) 2025/2026

This repository contains personal solutions and Python implementations for the Statistics (ESTAT) course, part of the Bachelor's in Computer Engineering (LEI) at the Instituto Superior de Engenharia do Porto (ISEP).

## Course Overview
The curriculum focuses on the application of statistical methods and introductory Python for engineering students.

### Core Topics
* **Chapter 1**: Introduction to Statistics and Python.
* **Chapter 2**: Descriptive Statistics (Autonomous study).
* **Chapter 3**: Probability Distributions (Discrete: Binomial and Poisson; Continuous: Uniform, Exponential, and Normal).
* **Chapter 4**: Sampling Distributions and Parameter Estimation (Central Limit Theorem and Confidence Intervals).
* **Chapter 5**: Hypothesis Testing (Means, Proportions, and Differences).
* **Chapter 6**: Simple Linear Regression (Autonomous study).
